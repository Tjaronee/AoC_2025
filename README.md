# Advent of Code 2025

My solutions for [Advent of Code 2025](https://adventofcode.com/2025).

## Setup

Install the utilities package:
```bash
pip install -e .
```

Install development dependencies:
```bash
pip install -r requirements-dev.txt
```

## Usage

In notebooks or Python scripts:
```python
from aoc_utils import getInput

data = getInput(1)  # Load Day 1 input
```

## Structure

- `src/aoc_utils/` - Utility functions
- `Python/` - Jupyter notebooks with solutions
- `Inputs/` - Input data files
- `tests/` - Unit tests
